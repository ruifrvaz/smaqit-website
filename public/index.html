smaQit

Spec-driven agent orchestration for greenfield projects with heavy business requirements. Describe requirements, specification agents generate stateful specs, then implementation agents turn those specs into working, tested, deployed applications.

  • Best fit for new builds carrying real business logic
  • Stateful specs — draft → implemented → deployed → validated
  • Bounded agents — one agent, one layer, no scope creep
  • Five layers, three phases, mapped to Agile roles
View smaQit on GitHub

smaQit Extensions

Adds tooling to an existing smaQit project — or runs standalone on application-heavy builds with little to no business logic. Quality-of-life workflows, agents, and skills for session management, task tracking, and release automation.

  • Pairs with smaQit to add tooling to a business-heavy project
  • Or runs standalone for app-heavy, logic-light builds
  • Session continuity, task tracking, release automation
  • Works in any git repository
View smaQit Extensions on GitHub

smaQit ADK

The complement that lets you build an agentic ecosystem that fits your own needs. Everything required to create custom Copilot agents and skills — smaQit itself is one example of what you can build with it.

  • Build your own agents and skills from scratch
  • Zero-config lite tier — just say "create a new agent"
  • Isolated CLI sessions for framework builders
  • Traceable L0 → L1 → L2 compilation chain
